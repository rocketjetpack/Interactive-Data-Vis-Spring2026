Do some transfrmation to get a new, different dataset

```js
const countsByIsland = {};

penguins.forEach((p) => {
    if (p.island in countsByIsland) {
        countsByIsland[p.island] += 1;
    } else {
        countsByIsland[p.island] = 1;
    }
});
```

```js
const data = Object.entries(countsByIsland).map(([island, count]) => ({ island, count}));
display(data);
```
